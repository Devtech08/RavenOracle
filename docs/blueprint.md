# **App Name**: Raven Oracle

## Core Features:

- Gateway Authentication: Require users to type 'raven.oracle' to access the website.
- Access Code Verification: Verify a secret access code before granting access to the chat room.
- End-to-End Encryption: Provide end-to-end encrypted messaging for secure communication.
- Message Auto-Purge: Automatically delete messages after a communication session ends for non-admin users.
- Admin Message Review: Allow administrators to access and review past message logs.

## Style Guidelines:

- Primary color: Dark purple (#1A1025) for a mysterious and secure feel.
- Background color: Deep black (#0A0A0F) to enhance the moody and dark atmosphere.
- Accent color: Electric cyan (#00FFFF) to highlight interactive elements and add a subtle glow.
- Font: 'Source Code Pro' monospace font to provide a secure terminal aesthetic. Note: currently only Google Fonts are supported.
- Use smooth fade and slide animations for a cinematic and modern UI.
- Implement a minimal, cinematic UI with subtle glowing effects for an immersive user experience.
